# REST API CRUD - GOLANG

---

## Stack :

- Golang (Gofiber)

---

## Package :

- github.com/gofiber/fiber/v2
- github.com/joho/godotenv
- gorm.io/driver/mysql
- gorm.io/gorm

---

## How to run this server :

- git clone
- cd go-fiber-crud
- create database mysql name "crudgolang"
- start server with command `go run main.go`
